# 🛒 Mi Tienda
Aplicación desarrollada en **Flutter** con integración de **Firebase**.

---

## 🚀 Tecnologías
- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore

---

## 🔍 Características
✔ Registro e inicio de sesión  
✔ Lista de productos  
✔ Favoritos  
✔ Carrito de compras  
✔ Perfil de usuario
